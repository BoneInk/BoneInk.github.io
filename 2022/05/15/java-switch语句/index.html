switch 语法格式：

switch 匹配的表达式可以是：

  • byte、short、char、int类型及 这4种类型的包装类型；
  • 枚举类型；
  • String 类型；

case 匹配的表达式可以是：

  • 常量表达式；
  • 枚举常量；

注意一点： case提供了switch表达式的入口地址，一旦switch表达式与某个case分支匹配，则从该分支的语句开始执行，一直执行下去，即其后的所有case分支的语句也会被执行，直到遇到break语句。
