数组类型

简介

在java中，数组也是一种引用类型，即是一种类。

输出的结果：

[I
[Ljava.lang.Integer;
[[Ljava.lang.Integer;
[F
[Ljava.lang.Float;

“[ ” 表示一维数组，“[[”则表示二维数组，如此类推下去。同时，如果是引用类型，则还要“[”后面还要跟一个”L“+类的全限定名。而如果是基本类型，则只要跟对应的大写字母。

数组分类

数组类可以分类可以分成两类：

  • 基本类型的数组类；
  • 引用类型的数组类；
    这两种数组类的最大区别在于他们的祖先类不同。

对于基本类型来说，基本类型数组类的父类一个，就是Object类。

运行结果：

int[]的superClass是： class java.lang.Object
